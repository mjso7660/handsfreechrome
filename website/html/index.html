Hands Free for Chrome

A Chrome extension to browse the web using only your voice, free to download and use.

Add to Chrome

You're not using Google Chrome! Once you're in Chrome, come back to download Hands Free here, or get it from the Chrome Web Store.

Quick Start Guide
After installing Hands Free, restart Chrome, then open any webpage and hit Ctrl + Space.
Give the input window permission to use your mic.
Then just say help.

Are you using a Mac?
In order to effectively use Hands Free on a Mac, first go to System Preferences, then to Sound, then to Input. If there's a little checkbox option that says "Use ambient noise reduction," make sure that it is not checked.


Hands Free can scroll around, click things, navigate to particular webpages, go back or forward in your browsing history, enter text into forms, check and uncheck checkboxes, zoom in and out, open and close tabs, refresh the page, and more.



This extension is for the Chrome web browser, which can be downloaded for free from the Google Chrome website.

Hands Free is open source, and the code is available on GitHub. If you would like to contribute to its development, take a look at the development page.
