Features to be added in the future include:


  • An improved version of the map command that supports more pages.
  • Phonetic mode for dictation mode, allowing you to use the International Phonetic Alphabet (alpha bravo charlie...) to spell things out. That way, you can literally write anything, even complex usernames and passwords. There would be a pop-up guide to the IPA to assist users. It's actually not too hard to remember.
  • A "find" feature to search pages for text, just like Ctrl+F.
  • A "tab map" command to number all the tabs, so you can pick one instantly instead of having to say "switch" repeatedly.
  • A visual indication of whether you're in dictation mode or control mode.
  • A way to view your bookmarks and browsing history, and to select pages from them.
  • Multiple window support. There would be a new command added to let you change which window you're controlling.
  • Once there's support for multiple windows, we can add a "split screen" command to automatically create two windows and position them so they each take up half the screen.
  • Select/Copy/Paste support.
  • A way to select options from dropdown menus.
  • Mulilingual support-- this is very hard. All commands must be carefully tested, with the most common mishearings manually hardcoded, so that for example saying "two" to click a number tag does not result in the speech recognition hearing "to" or "too" instead and nothing happening. Making the extension multilingual would be a fairly large undertaking.
